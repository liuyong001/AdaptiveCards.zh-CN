# <a name="native-styling---javascript"></a>本机样式-JavaScript

使用 CSS 进行细粒度的卡 HTML 样式。

以下的 CSS 类将添加到各种元素。

| CSS 类 | 用法 |
|---|---|
| .ac-container | 容器 |
| .ac-selectable  | 具有的元素 `selectAction` |
| .ac-image | image |
| .ac-pushButton | 呈现按钮等的操作 |
| .ac-linkButton  | 呈现链接等操作 |
| .ac-input | 输入的控件|
| .ac-textInput| 文本输入 |
| .ac-multiline | 多行文本输入 |
| .ac-numberInput | 数字输入|
| .ac-dateInput | 日期输入|
| .ac-timeInput | 时间输入 |
| .ac-multichoiceInput | 单选输入|
