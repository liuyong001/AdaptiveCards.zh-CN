# <a name="native-styling---javascript"></a>本机样式设置-JavaScript

使用 CSS 对卡片 HTML 进行精细的样式设置。

以下 CSS 类将添加到各种元素。

| CSS 类 | 用法 |
|---|---|
| .ac-container | 存放 |
| . 交流电-可选  | 元素与`selectAction` |
| .ac-image | image |
| .ac-pushButton | 呈现的操作类似于按钮 |
| .ac-linkButton  | 呈现的操作, 如链接 |
| .ac-input | 输入控件|
| .ac-textInput| 文本输入 |
| .ac-multiline | 多行文本输入 |
| .ac-numberInput | 数字输入|
| .ac-dateInput | 日期输入|
| .ac-timeInput | 时间输入 |
| .ac-multichoiceInput | multichoice 输入|
