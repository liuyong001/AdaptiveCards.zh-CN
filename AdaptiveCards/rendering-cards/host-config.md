# <a name="what-is-hostconfig"></a><span data-ttu-id="74b25-102">什么是 HostConfig？</span><span class="sxs-lookup"><span data-stu-id="74b25-102">What is HostConfig?</span></span>
<span data-ttu-id="74b25-103">`HostConfig` 是一个**跨平台配置对象**，用于指定自适应卡片呈现器生成 UI 的方式。</span><span class="sxs-lookup"><span data-stu-id="74b25-103">`HostConfig` is a **cross-platform configuration object** that specifies how an Adaptive Card Renderer generates UI.</span></span>
