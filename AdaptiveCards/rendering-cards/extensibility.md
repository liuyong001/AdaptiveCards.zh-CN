# <a name="extensibility"></a>扩展性

每个 SDK 都允许你重写任何元素的呈现，甚至允许你为自己定义的全新元素添加支持。  例如，可以更改 `Input.Date` 呈现器，以便在仍然保留呈现器的其余输出的情况下，发出你自己的自定义控制。 也可添加对你定义的自定义 `Rating` 元素的支持。

如需示例代码，请展开左侧的 **SDK** 节点，然后单击“呈现卡片”   -> “要使用的 SDK”   ->   “可扩展性”
