# <a name="extensibility"></a>扩展性

每个 SDK，可重写的呈现任何元素，或者甚至添加对您定义的整个新元素的支持。  例如，可以更改`Input.Date`呈现器仍保留的呈现器输出的其余部分的同时发出自定义控件。 也可以添加对自定义支持`Rating`给您的元素定义。

有关代码示例，请展开**SDK**左侧的节点->**呈现卡** -> **你想要使用的 SDK**  ->  **可扩展性**
