# <a name="partners"></a>合作伙伴 

自适应卡在受益于广泛的开发人员时最有用。 我们的愿景是创建一个共享生态系统, 开发人员可在其中在多个平台上编写和共享内容。 我们的目标是与社区和公司合作来构建这一构想, 并开发通用卡交换框架。

如果你有兴趣加入自适应卡生态系统, 请[直接在 GitHub 上联系](https://github.com/Microsoft/AdaptiveCards)。
